# 🌿 Plant Disease Detector for Farmers

An AI-powered web application that helps farmers identify plant diseases using their smartphone camera.

## 🚀 Features

- 📸 Real-time camera capture
- 🤖 AI-powered disease identification
- 💊 Chemical and organic treatment recommendations
- 📱 Mobile-responsive design
- 🆓 Free and open-source

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- TensorFlow.js
- MobileNet Model

## 📋 How to Use

1. Open the app in a modern browser (Chrome, Firefox, Safari)
2. Click "Start Camera" and allow camera access
3. Take a clear photo of the affected plant leaf
4. Click "Capture & Identify" or upload a photo
5. View the diagnosis and treatment recommendations

## 🚀 Quick Start (Local)

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/plant-disease-detector.git

# Navigate to project folder
cd plant-disease-detector

# Start a local server (Python 3)
python -m http.server 8000

# Or with Node.js
npx serve

# Open browser and visit: http://localhost:8000
